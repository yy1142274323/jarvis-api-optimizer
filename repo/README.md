## 中文说明 🇨🇳

这是一个专为 OpenClaw 和其他 LLM 代理设计的**语义缓存层**。通过智能识别重复或相似的请求，它能为您节省高达 30% 的 Token 损耗。

### 核心功能
- **语义哈希**：智能识别相似查询，直接返回缓存结果。
- **SQLite 后端**：轻量、极速、零配置。
- **一键集成**：轻松接入您的助手请求循环。

### 安装方式
1. 克隆仓库：`git clone https://github.com/yy1142274323/jarvis-api-optimizer.git`

## English Description 🇺🇸

A semantic caching layer for OpenClaw and other LLM agents. Reduce your API token costs by up to 30% by skipping redundant requests.

### Features
- **Semantic Hashing**: Identifies similar queries and returns cached responses.
- **SQLite Backend**: Lightweight and fast.
- **Easy Integration**: Plug it into your agent's request loop.
